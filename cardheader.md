# &lt;CardHeader&gt;

A header inside a Card

![](/assets/CardHeader.png)

## Attributes

**title **string \(default: ""\)

A title for a card.

**subtitle** string \(default: ""\)

A subtitle for a card

## Examples

```js
<Card>
  <CardHeader title="Card title" subtitle="Card subtitle" />
</Card>
```

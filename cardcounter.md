# &lt;CardCounter&gt;

A counter card component

## Attributes

**number** number \(default: null\)

Counter value displayed on the card

## Examples

JSX
```js
<Card>
  <CardCounter number={28}/>
</Card>
```

```js
{
  "elementName": "Card",

    {
      "elementName": "CardCounter",
      "attributes": {

      }
    }
  ]
}
```
